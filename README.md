# 🏏 IPL Data Analysis Project

## 📌 Overview

This project focuses on analyzing Indian Premier League (IPL) match data using Python. The goal is to extract meaningful insights about team performance, match trends, and the impact of different factors such as toss decisions.

---

## 📊 Objectives

* Analyze IPL match datasets to understand team performance
* Study the impact of toss decisions on match outcomes
* Visualize data to identify patterns and trends
* Perform basic exploratory data analysis (EDA)

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib

---

## 📂 Dataset

The dataset used in this project contains IPL match details such as:

* Teams
* Match results
* Toss winners
* Runs scored
* Venues

---

## 🔍 Key Analysis Performed

* Top teams based on number of wins
* Toss winner vs match winner analysis
* Distribution of runs scored
* Team performance comparison

---

## 📈 Visualizations

* Pie charts for team wins
* Bar graphs for performance comparison
* Stacked plots for scoring patterns

---

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/mr7607/IPL-Analysis.git
```

2. Navigate to the project folder:

```bash
cd IPL-Analysis
```

3. Install required libraries:

```bash
pip install pandas matplotlib
```

4. Run the script:

```bash
python your_script_name.py
```
